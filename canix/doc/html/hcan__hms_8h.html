Dokumentation der Funktionen

void canix_SFP_HMS_handler ( const canix_frame *  frame  ) 

Stellt den HCAN Frame Callback dar. Es werden folgende Frames beantwortet bzw. interpretiert:

  • HCAN_HMS_PING_REQUEST
  • HCAN_HMS_DEVICE_STATE_REQUEST
  • HCAN_HMS_DEVICE_RESET
  • HCAN_HMS_DEVICE_TYPE_REQUEST
  • HCAN_HMS_INTERNAL_EEPROM_WRITE
  • HCAN_HMS_INTERNAL_EEPROM_READ_REQUEST
  • HCAN_HMS_UPTIME_QUERY
  • HCAN_HMS_DEBUG_LEVEL_SET
  • HCAN_HMS_BUILD_VERSION_QUERY

Definiert in Zeile 39 der Datei hcan_hms.c.

Benutzt can_error_counters(), canix_frame_send(), canix_reset(), canix_rtc_clock, canix_rx_lost, canix_rx_received, canix_selfaddr(), canix_set_syslog_level(), canix_tx_dropped, canix_tx_sent, canix_frame::data, canix_frame::dst, canix_frame::proto, canix_frame::size, canix_frame::src und canix_rtc_clock_t::uptime_sec.

Wird benutzt von canix_init().
