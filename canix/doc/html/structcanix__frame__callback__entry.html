Ausführliche Beschreibung

Canix stellt ein (H)CAN Interface zur Verfuegung. Jeder Client, der HCAN-Frames empfangen moechte, kann einen ueber canix_reg_frame_callback(..) registieren. Die Filter-Daten (Ziel, Protokoll, Service) werden zusammen mit der Adresse der Callback-Funktion in Instanzen dieser Struktur gespeichert.
